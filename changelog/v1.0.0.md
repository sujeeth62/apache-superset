# Changelog - v1.0.0

## [1.0.0] - 2025-11-12

### Added
- Initial release of the custom Apache Superset Docker image
- Based on official `apache/superset:5.0.0` image
- Added support for multiple database drivers:
  - Oracle
  - PostgreSQL (psycopg2-binary)
  - MySQL (mysqlclient)
  - Microsoft SQL Server (pymssql)
  - Google BigQuery (sqlalchemy-bigquery)
  - Elasticsearch (elasticsearch-dpapi)
  - Apache Solr (sqlalchemy-solr)
- Integrated Playwright for Alerts & Reports
- Added OpenPyXL for Excel file uploads
- Included Pillow for image processing
- Optimized Docker image for production use
- GitHub Actions workflow for CI/CD
- Comprehensive documentation in README.md

### Changed
- Updated default configuration for production use
- Improved security settings and defaults
- Enhanced Dockerfile for better layer caching and smaller image size

### Security
- Implemented secure defaults for production deployment
- Added guidance for secure configuration
- Included security best practices in documentation
